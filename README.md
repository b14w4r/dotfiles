# dotfiles

Arch Linux + Hyprland dotfiles.

## Установка пакетов

```bash
sudo pacman -S hyprland hyprpaper hypridle hyprlock xdg-desktop-portal-hyprland waybar rofi-wayland swaync wl-clipboard jq polkit-gnome network-manager-applet blueman pavucontrol playerctl brightnessctl power-profiles-daemon grim slurp qt5ct qt6ct telegram-desktop obsidian ghostty
```

## AUR (через yay или paru)

```bash
yay -S wlogout waypaper cliphist hyprshade
```
